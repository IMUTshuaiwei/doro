# Doro 开发文档

## 项目简介

Doro 是一个桌面宠物应用，提供丰富的交互功能，包括动画展示、系统信息显示、随机移动等。

## 安装步骤

1. 克隆项目代码：

   ```bash
   git clone https://github.com/IMUTshuaiwei/doro
   ```

2. 进入项目目录：

   ```bash
   cd doro
   ```

3. 安装依赖：

   ```bash
   pip install -r requirements.txt
   ```

## 运行方法

1. 启动应用：

   ```bash
   python main.py
   ```

2. 确保资源文件夹 `resources` 存在并包含必要的配置和动画文件。
